# Dispatching-Parallel-Agents 技能使用完全指南

> 来源：obra/superpowers 插件 v5.0.7
> 整理：2026-05-05

## 概述

每个独立问题域分发一个代理。让它们同时工作。

**核心原则：** 当有多个无关失败时，并行调查节省时间。

## 何时使用

- 3+ 测试文件失败，不同根因
- 多个子系统独立损坏
- 每个问题可在无其他上下文的情况下理解
- 调查之间无共享状态

## 模式

1. **识别独立域** — 按破坏的东西分组失败
2. **创建专注代理任务** — 每个代理获得具体范围和约束
3. **并行分发** — Task() 三个代理同时运行
4. **审查和集成** — 检查冲突，运行完整测试套件

## 代理 Prompt 结构

好的代理 prompt：
- **专注** — 一个清晰的问题域
- **自包含** — 理解问题所需的全部上下文
- **具体关于输出** — 代理应该返回什么？
