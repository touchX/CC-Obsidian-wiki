# diagram-design

Thirteen editorial diagram types for Claude Code. Self-contained HTML + SVG with no shadows or Mermaid-style clutter.

## 核心特点

| 特性 | 描述 |
|------|------|
| **13 种图表类型** | 编辑级别的设计图表 |
| **纯 HTML + SVG** | 自包含，无依赖 |
| **无阴影设计** | 简洁干净的视觉风格 |
| **Claude Code 技能** | 专为 Claude Code 设计 |

## 与 Mermaid 的区别

- **自包含 SVG** vs Mermaid 需要渲染器
- **编辑级设计** vs 自动生成样式
- **无阴影/无杂讯** vs Mermaid 默认样式
- **直接嵌入** vs 外部依赖

## 安装

```bash
npx skills add cathrynlavery/diagram-design
```

## 使用方式

Claude Code 中直接描述需要的图表类型，skill 会生成对应的 HTML + SVG。
