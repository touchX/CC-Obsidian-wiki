## 项目介绍

Next.js 是由 Vercel 开发的 React 框架，用于构建生产级应用。它提供服务端渲染 (SSR)、静态站点生成 (SSG)、API 路由、Codemods 等功能，是现代 Web 开发的主流选择。

### 核心特性

- **App Router** — 基于 React Server Components 的新一代路由系统
- **Pages Router** — 传统页面路由（仍支持）
- **Turbopack** — 极速打包工具（Rust 编写）
- **TypeScript** — 原生 TypeScript 支持
- **API Routes** — 轻松创建 API 端点
- **图像优化** — next/image 自动优化图片
- **国际化** — 内置 i18n 路由支持

## 安装与使用

```bash
# 创建新项目
npx create-next-app@latest my-app

# 手动安装
npm install next react react-dom

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
