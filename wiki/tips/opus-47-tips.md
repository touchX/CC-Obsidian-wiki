| 技巧 | 功能 |
|------|------|
| Auto Mode | 模型分类器自动审批安全命令 |

| Recaps | 总结 Agent 工作内容和下一步 |
| Focus Mode | 隐藏中间过程只看最终结果 |
| Effort | 五级调节：low → max |
| 验证机制 | 让 Claude 有方式验证工作 |
