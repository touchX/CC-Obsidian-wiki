# Code Review & Test Time Compute

## 核心要点

| 技巧 | 说明 |
|------|------|
| Code Review | PR 打开时自动派遣 Agent 团队深度审查 |
| Test Time Compute | 更多 token = 更好结果；分离 context window 效果更优 |
