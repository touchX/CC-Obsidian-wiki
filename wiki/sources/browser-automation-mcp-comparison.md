# Browser Automation MCP Comparison

三大浏览器自动化 MCP 工具全面对比：Chrome DevTools MCP（Google 官方）、Claude in Chrome（Anthropic 官方）和 Playwright。从截图能力、执行速度、API 设计和 token 消耗四个维度评估各工具优劣与适用场景。
