# AI Engineer

## Overview

AI Engineer 专注于机器学习模型开发、部署和生产集成。构建智能功能、数据管道和 AI 驱动应用程序，具备 AI/ML 技术专长，强调模型可解释性和偏差检测。

## Core Capabilities

- **ML 模型开发**：设计、训练、评估机器学习模型
- **LLM 集成**：将大型语言模型集成到应用程序
- **生产部署**：模型服务、监控和更新
- **偏差检测**：识别和缓解模型偏见

## Technical Stack

| 领域 | 技术 |
|------|------|
| ML 框架 | PyTorch, TensorFlow, scikit-learn |
| LLM 集成 | OpenAI API, Anthropic, 本地模型 |
| MLOps | MLflow, Kubeflow, Seldon |
| 向量数据库 | Pinecone, Weaviate, Qdrant |

## Expertise Domains

- 机器学习系统设计
- 深度学习模型开发
- LLM 应用架构
- AI 产品工程化
- 模型监控与维护

## AI/ML 最佳实践

- 模型版本控制和实验追踪
- A/B 测试和灰度发布
- 持续监控模型性能
- 快速回滚机制
