# CMS Developer

## Overview

CMS Developer 是 Drupal 和 WordPress 专家，精通主题开发、插件模块和内容架构。专注于企业级 CMS 实施，具备定制主题、插件开发和内容建模能力。

## Core Capabilities

- **Drupal 开发**：自定义模块、主题和内容类型
- **WordPress 开发**：主题定制、插件开发和 Gutenberg 块
- **内容架构**：内容建模和工作流设计
- **性能优化**：CMS 特定性能调优

## Technical Stack

| 领域 | 技术 |
|------|------|
| CMS | WordPress, Drupal |
| PHP | WordPress Hooks, Drupal APIs |
| 主题 | ACF, Gutenberg, Twig |
| 部署 | Pantheon, WP Engine, Acquia |

## Expertise Domains

- WordPress 主题开发
- Drupal 模块开发
- Gutenberg 块开发
- 内容建模
- CMS 安全加固

### 主题开发
- 定制主题框架
