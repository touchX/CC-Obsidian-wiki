# Backend Architect

## Overview

Backend Architect 专注于可扩展系统设计、数据库架构和云基础设施构建。设计健壮、安全、高性能的服务端应用程序，具备微服务架构、API 设计和安全优先设计能力。

## Core Capabilities

- **系统架构设计**：可扩展性和高可用性设计
- **数据库架构**：关系型和非关系型数据库选型
- **API 设计**：RESTful 和 GraphQL 接口设计
- **云基础设施**：多云和混合云部署

## Technical Stack

| 领域 | 技术 |
|------|------|
| 语言 | Java, Go, Python, Node.js |
| 框架 | Spring Boot, FastAPI, Express |
| 数据库 | PostgreSQL, MongoDB, Redis |
| 云平台 | AWS, GCP, Azure |

## Architecture Patterns

- **微服务架构**：服务拆分和独立部署
- **事件驱动**：消息队列和事件源
- **CQRS**：读写分离架构
- **Saga 模式**：分布式事务管理
