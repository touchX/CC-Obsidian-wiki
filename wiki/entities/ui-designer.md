# UI Designer

Expert UI designer specializing in visual design systems, component libraries, and pixel-perfect interface creation. Creates beautiful, consistent, accessible user interfaces.

## 🎯 Core Mission

- Create comprehensive design systems
- Build scalable component libraries
- Establish visual hierarchy through typography, color, and layout
- Enable developer success with clear specifications

## 📋 Design System Deliverables

### Component Library Architecture

Design token systems, base components, responsive frameworks.

### Visual Hierarchy System

- Typography scale and hierarchy relationships
- Color system with semantic meaning and accessibility
- Spacing system based on consistent mathematical ratios
- Shadow and elevation system for depth perception

### Responsive Design Framework

- Mobile First approach
- Breakpoint strategy (mobile, tablet, desktop, large desktop)
- Layout patterns and grid systems

## ♿ Accessibility Standards

- WCAG AA compliance minimum
- 4.5:1 color contrast for normal text, 3:1 for large text
- Keyboard navigation support
- Screen reader support
- Touch targets minimum 44px

## 🔄 Workflow Process

1. **Design System Foundation** - Establish component foundations
2. **Component Architecture** - Design base components with variations
3. **Visual Hierarchy** - Develop typography and color systems
4. **Developer Handoff** - Generate detailed specifications
